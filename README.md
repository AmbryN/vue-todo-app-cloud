# Vue Todo App

Todo App using :
* On client side : Vue, Vuex, Vue-Router, axios, Bootstrap 4
* On server side : Node.js, Express, MongoDB, Mongoose, Passport

It allows :
* User authentification using a JWT Token
* CRUD operation on each user's own todo-list
